# Console端接口文档

## BaseURL：

`http://localhost:8081`

## 新增

### /product/create

### 参数

  images: [String],
  title: [String],
  price: [Float],
  description: [String]

### 返回

成功or失败

## 修改

### /product/update

### 参数

  id: [Long],
  images: [String],
  title: [String],
  price: [Float],
  description: [String]

### 返回

成功or失败

## 删除

### /product/delete

### 参数:

id:[Long]

### 返回

成功or失败

## 数据库建表语句

```sql
create table product(
id bigint unsigned not null auto_increment,
images varchar(2000) character set utf8mb4 collate utf8mb4_unicode_ci not null comment '商品图片列表',
title varchar(50) character set utf8mb4 collate utf8mb4_unicode_ci not null comment '商品标题',
price decimal(10,2) unsigned not null comment '商品价格',
description varchar(255) character set utf8mb4 collate utf8mb4_unicode_ci comment '商品描述',
create_time int unsigned not null,
update_time int unsigned not null,
is_deleted tinyint unsigned not null default 0,
primary key (id),
key idx_title (title)
) engine=innodb default charset=utf8mb4 collate=utf8mb4_unicode_ci comment='产品表';
```
